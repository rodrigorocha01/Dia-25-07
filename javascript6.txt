DO-WHILE == O laço Do-While serve para executar um comando enquanto a condição não é cumprida
A diferença entre o laço   WHILE e o laço DO-WHILE é que no laço DO - WHILE ele vai executar o código pelo
menos uma vez para depois ver se a condição está certa
while( condição ){ código }
Ele só vai executar o código enquanto a condição não é cumprida
do{ código }while(condição) 
Ele executa o código pelomenos uma vez para depois ver a condição
